ℹ️ About This Project

The Perfect Bracket by Owen Soeder is a March Madness experiment where 50,000 unique brackets were generated using AI-powered seed probability modeling — all before the first game tipped off on March 17, 2026.

Each bracket uses historical seed win rate data. For every matchup, the algorithm weights the probability based on seed, then applies randomness so all 50,000 brackets are unique but statistically plausible.

Built with Claude by Anthropic. All brackets were generated and published on Selection Sunday, March 15, 2026 — before any games were played.

Methodology
• 50,000 unique brackets via Python simulation
• Seed win probabilities from historical NCAA data
• 1-seeds win ~98.5% of first round games
• 12-seeds upset 5-seeds ~35% historically
• Every bracket independently simulated — no two identical
• Thanks to Claude by Anthropic 🤖
